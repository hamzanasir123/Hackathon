## Overview

This example demonstrates the basic setup of loading a humanoid robot model in Isaac Sim with a simple scene configuration. This is the foundation for all other Isaac Sim examples.

## Learning Objectives

After completing this example, you will be able to:

- Launch Isaac Sim with a humanoid robot model
- Configure basic scene elements (ground plane, lighting)
- Set initial robot pose and configuration
- Verify basic simulation physics

## Setting Up the Basic Scene

### Python Setup Script

<CodeExample
  language="python"
  title="Basic Isaac Sim Setup Script"
  description="Python script to set up basic humanoid simulation"
  code={`#!/usr/bin/env python3
import omni
from omni.isaac.core import World
from omni.isaac.core.utils.nucleus import get_assets_root_path
from omni.isaac.core.utils.stage import add_reference_to_stage
from omni.isaac.core.utils.prims import set_targets
from pxr import Gf
import numpy as np

def setup_basic_humanoid_simulation():
    """
    Set up a basic humanoid robot simulation in Isaac Sim
    """
    # Create a simulation world
    my_world = World(stage_units_in_meters=1.0)

    # Get assets root path
    assets_root_path = get_assets_root_path()
    if assets_root_path is None:
        print("Could not use Isaac Sim assets. Ensure Isaac Sim is installed.")
        return None

    # Add a ground plane
    add_reference_to_stage(
        usd_path=assets_root_path + "/Isaac/Environments/Simple_Room/simple_room.usd",
        prim_path="/World/GroundPlane"
    )

    # Load a humanoid robot asset
    # Note: This is a simplified example - actual humanoid assets may vary
    humanoid_asset_path = assets_root_path + "/Isaac/Robots/Humanoid/humanoid.usd"
    add_reference_to_stage(
        usd_path=humanoid_asset_path,
        prim_path="/World/Humanoid"
    )

    # Set initial pose for the humanoid
    from omni.isaac.core.utils.transformations import set_position_orientation
    set_position_orientation(
        prim_path="/World/Humanoid",
        position=np.array([0.0, 0.0, 1.0]),
        orientation=np.array([1.0, 0.0, 0.0, 0.0])  # No rotation (w, x, y, z)
    )

    # Add lighting
    from omni.isaac.core.utils.prims import create_prim
    create_prim(
        prim_path="/World/Light",
        prim_type="DomeLight",
        position=np.array([0, 0, 5]),
        attributes={"inputs:intensity": 3000}
    )

    # Reset the world
    my_world.reset()

    print("Basic humanoid simulation setup complete!")
    print("Robot loaded at position: [0.0, 0.0, 1.0]")
    print("Simulation ready for physics testing")

    return my_world

def run_basic_simulation():
    """
    Run the basic simulation for a number of steps
    """
    world = setup_basic_humanoid_simulation()
    if world is None:
        return

    print("Running basic simulation for 1000 steps...")

    for i in range(1000):
        # Step the world
        world.step(render=True)

        # Print progress every 200 steps
        if i % 200 == 0:
            print(f"Simulation step: {i}/1000")

    print("Basic simulation completed!")

if __name__ == "__main__":
    run_basic_simulation()`}
/>

## Verification Steps

1. **Launch Isaac Sim** with the basic humanoid scene
2. **Verify robot model** loads correctly in the scene
3. **Check physics simulation** - robot should maintain its initial position
4. **Confirm rendering** - scene should be properly illuminated
5. **Validate initial pose** - robot should be positioned as specified
