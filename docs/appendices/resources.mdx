# Resources and Tools

This appendix provides a collection of resources, tools, and references to support your learning journey in Physical AI and humanoid robotics.

## Software Tools

### Development Environments
- [ROS 2 Humble Hawksbill](https://docs.ros.org/en/humble/) - Current LTS version of Robot Operating System
- [NVIDIA Isaac Sim](https://developer.nvidia.com/isaac-sim) - GPU-accelerated simulator for robotics
- [Gazebo Garden](http://gazebosim.org/) - Open source robotics simulator
- [Unity Robotics Hub](https://unity.com/solutions/industrial-automation/robotics) - Unity tools for robotics

### Programming Languages
- [Python 3.8+](https://www.python.org/) - Primary language for robotics development
- [C++17](https://isocpp.org/) - Performance-critical robotics applications
- [TypeScript](https://www.typescriptlang.org/) - For web-based robotics interfaces

## Frameworks and Libraries

### AI and Machine Learning
- [PyTorch](https://pytorch.org/) - Deep learning framework
- [TensorFlow](https://www.tensorflow.org/) - ML library
- [OpenCV](https://opencv.org/) - Computer vision library
- [PCL (Point Cloud Library)](https://pointclouds.org/) - 3D point cloud processing

### Control and Planning
- [MoveIt](https://moveit.ros.org/) - Motion planning framework
- [OMPL](https://ompl.kavrakilab.org/) - Open Motion Planning Library
- [control toolbox](https://github.com/ros-controls/ros2_control) - ROS 2 control framework

## Research Papers and Publications

### Key Papers in Physical AI
- "Physical Intelligence" - Overview of the field
- "Learning Dexterous Manipulation from Random Object Grasps" - Deep learning for manipulation
- "End-to-End Learning of Action Selection for Robot Grasping" - Reinforcement learning for robotics

## Books and Textbooks

### Recommended Reading
- "Robotics, Vision and Control" by Peter Corke
- "Probabilistic Robotics" by Sebastian Thrun, Wolfram Burgard, and Dieter Fox
- "Springer Handbook of Robotics" by Bruno Siciliano and Oussama Khatib
